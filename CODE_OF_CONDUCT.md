# Code of Conduct

## Our standard

This project values directness, honesty, and practical contribution.

Treat other contributors like adults. Be respectful, be specific, and be useful.

## Expected behavior

- Be clear.
- Critique the code, not the person.
- Bring evidence when you disagree.
- Prefer reproducible reports over drama.
- Keep the project focused on shipping something that works.

## Unacceptable behavior

- Personal attacks
- Harassment or threats
- Intentionally misleading bug reports
- Spam, trolling, or low-effort disruption
- Posting private information without permission

## Enforcement

Project maintainers may remove content, close discussions, or block participation when behavior harms the project or wastes contributor time.
