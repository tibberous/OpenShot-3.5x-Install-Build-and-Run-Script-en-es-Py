# Security Policy

## Supported versions

The current maintained line is:

- `1.0` — supported for active fixes and polish

## Reporting a vulnerability

Please do **not** post security issues as public bug reports first.

Report security concerns directly to:

- **Trenton Tompkins**
- **Email:** trenttompkins@gmail.com
- **Website:** https://www.trentontompkins.com

When possible, include:

- affected version
- proof of concept or steps to reproduce
- impact summary
- whether the issue affects build, bootstrap, packaging, or generated artifacts

You will receive a response as quickly as practical. If the issue is valid, the goal is to fix it cleanly and document the remediation without leaking unnecessary exploit detail.
