OpenShot BnR 1.0
================

OpenShot BnR is a Windows install, build, run, and distro-prep script for OpenShot 3.5.x.

What it does:
- checks prerequisites
- resolves live MSYS2 dependencies
- clones or updates OpenShot repos
- builds libopenshot-audio and libopenshot
- verifies bindings honestly
- generates launch, portable, and frozen helpers
- exposes docs and debug output from the command line

Quick start:
  py -3 OpenShot_BnR_v1_0.py

Useful commands:
  py -3 OpenShot_BnR_v1_0.py --usage
  py -3 OpenShot_BnR_v1_0.py --about
  py -3 OpenShot_BnR_v1_0.py --version
  py -3 OpenShot_BnR_v1_0.py --readme
  py -3 OpenShot_BnR_v1_0.py --license
  py -3 OpenShot_BnR_v1_0.py --debug
